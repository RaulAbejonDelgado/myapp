Instalacion de Angular

Requisitos:

*- Tener instalado NodeJs :NodeJS
Importante: Para comprobar la version de node y npm(Gestor de paquetes de node), abrimos una terminal y ejecutamos los siquientes comandos:
node -v Nos muestra la version de node.
npm -v Nos muestra la version del gestor de paquetes node.

Instalacion del cliente

*- Con NodeJs instalado, desde una terminal instalamos el cliente de Angular, con el siguiente comando.
npm install -g @angular/cli

Arrancar el primer proyecto Angular en desrrollo

En este momento estamos listos para ubicarnos en la ruta donde decidamos iniciar nuestros proyecto.

Desde la terminal navegamos hasta la ubicacion. Por ejemplo cd c: , mkdir proyecto ,cd proyecto,ng new my-app

Despues de que termine el proceso tras ejecutar el ultimo comando, podemos iniciarlo con ng serve -o con lo que si todo a ido bien, nos abrira una ventana de nuestro navegador con nuestra primera pagina angular. la ruta de acceso es http://localhost:4200
