Los componentes en Angular !!

En Angular se desarrolla en base a componentes. Los componentes son una vista con su logica asociada a una etiqueta que puede ser instanciada desde otros componentes o desde el sistema de rutas.

Crear un componente

comando a usar : ng generate component/nombre-componente

  • Forma abreviada de declarar un componente : ng g c component/nombre-componente
  • Los componentes a usar deben ser declarados en app.modulo.ts
  • Si quieres reusar el componente en otro componente debes usar el 'selector' del componente en la vista mediante una etiqueta
  • Declararlo en el app.routing.module.ts para activarlo en el sistema de rutas

Archivos resultantes tras la ejecucion del comando:

nombre-componente.componente.html :La vista del componente, donde puede ser una maquetacion estatica, o sacar datos de forma dinamica .

nombre-componente.component.ts :El controlador La logica del componente , las datos instaciados aqui tras declararlos e inicializarlos pueden ser accedidos desde la vista.

nombre-componente.component.scss :Donde se declaran los estilos, que se aplicaran en la vista. Al crear el proyecto en la version actual de angular cli, te pregunta sobre que tipo de estilos quieres usar, dependiendo la eleccion en este punto podemos encontrarnos con diferentes extensiones como css.. etc.

nombre-componente.component.spec.ts :No tocar xD.
